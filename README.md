# Bike Sales Analysis with Tableau

This project contains visualization and preparing the dashboard of bike sales analysis with Tableau


## Features
- Dataset contains 113037 records and 18 columns which (Date,Day,Month,Year,Customer_Age,	Age_Group,Customer_Gender,Country,State,Product_Category,Sub_Category,Product,Order_Quantity	Unit_Cost,Unit_Price,Profit	Cost,Revenue)
- Dashboard has parameters such as "Year" and "Country" on the left side.
- According to these parameters, tables will change automatically.
- On the left side , can be seen general profit , year to year changing of this profit, general revenue, how many units of which product were sold in this Bike Sales area.
- Dashboard has a bar chart which shows "Top 10 Products Sold by Quantity"
- Dashboard has a column chart which shows "Product Category Sales By Gender"
- Dashboard has a bubble chart which shows "Sales by Age Group and Product SubCategory"
- Dashboard has a map which shows "Sales by Country"
